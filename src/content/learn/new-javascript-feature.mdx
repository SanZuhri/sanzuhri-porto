# Exploring the Latest JavaScript Features

JavaScript is constantly evolving. Let's look at some of the exciting new additions.

## Optional Chaining (`?.`)

Simplifies accessing properties of objects that might be null or undefined.

```javascript
const user = { name: "Alice", address: { street: "123 Main" } };
const street = user?.address?.street; // "123 Main"
const city = user?.address?.city;   // undefined
```

## Nullish Coalescing (`??`)

Provides a default value only when the expression on the left is `null` or `undefined` (not for `0` or `''`).

```javascript
const foo = null ?? 'default string'; // "default string"
const bar = 0 ?? 42;                // 0
```

## Top-Level `await`

Allows using `await` keyword outside of async functions in modules.

```javascript
// module.js
const response = await fetch('https://api.example.com/data');
const data = await response.json();
console.log(data);
```

## Conclusion

These features make JavaScript more robust and enjoyable to write. Keep exploring!
